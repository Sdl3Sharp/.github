# SDL3#

![SDL3# Banner](https://raw.githubusercontent.com/Sdl3Sharp/Sdl3Sharp-assets/main/banner.png)

## About

**SDL3# is a collection of hand-crafted, idiomatic C# bindings for [SDL3](https://www.libsdl.org/) and its companion libraries**.

The goal is to provide C# developers with a binding that feels native to the language, rather than a thin auto-generated wrapper over the native API.

## Packages

The following packages are available:

- **[Sdl3Sharp](https://github.com/Sdl3Sharp/Sdl3Sharp)**: C# bindings for [SDL3](https://github.com/libsdl-org/SDL), the core library for window management, input, audio, rendering, and more.

More packages for SDL companion libraries are planned, including [SDL_image](https://github.com/libsdl-org/SDL_image), [SDL_ttf](https://github.com/libsdl-org/SDL_ttf), and [SDL_mixer](https://github.com/libsdl-org/SDL_mixer).
